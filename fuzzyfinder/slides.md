# fuzzyfinder

* quickly find the file you want
* works on buffers too
* and tags too
* and commands, lines, bookmarks, ...

!SLIDE bullets

# fuzzyfinder mappings

* <b>,ff</b> - find file
* <b>,fr</b> - find file by mru
* <b>,fl</b> - find file by mru in cwd
* <b>,fb</b> - find buffer
* <b>,ft</b> - find tag
* <b>,fc</b> - clear cache
