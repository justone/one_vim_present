!SLIDE bullets

# sessionman

* save and restore editing sessions
* improves on built-in :mksession
* session data stored in ~/.vim/sessions

!SLIDE bullets

# sessionman mappings

* <b>,ss</b> - save session (will prompt if new)
* <b>,sl</b> - list sessions
* <b>,so</b> - open session
* <b>,sc</b> - close session
* <b>,sa</b> - save session as
